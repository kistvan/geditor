*インストール方法

- .Net Framework
- Slim DX
- DirectX 9c
を使用しています。
Slim DXはランタイムが必要なので
http://slimdx.org/download.php
から.Net Frameworkのバージョンに応じたものをインストールしてください。
.Net Frameworkのバージョン確認は、
http://www.vector.co.jp/soft/win95/util/se489653.html
などを使ってください。


*アンインストール方法

このアプリケーションをフォルダごと削除してください。レジストリ等は使用していません。

*操作方法

回転
	3D画面上で右ボタンを押しながらドラッグ
ズーム
	マウスホイールを回転

カメラ位置をリセット
	マウスホイールをダブルクリック


** フォルダ説明

projects - 開発中のゲームデータや素材の保存先
	sample - サンプルプロジェクトの保存先
		サンプルプロジェクトで使用する3Dモデルの保存先。floor.mqoを変更すると読み込まれる3Dモデルも変更されます。
resources - エディタが使用する素材などの保存先。変更しないでください。
